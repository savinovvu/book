## О чём задача?

Задача "Permutation in String" (Перестановка в строке) проверяет, 
содержится ли любая перестановка строки s1 в строке s2 в качестве подстроки.

Пример:

s1 = "ab", s2 = "eidbaooo"

Результат: true, потому что перестановка "ba" содержится в s2


## Решение на Java

```java
class Solution {
    public boolean checkInclusion(String s1, String s2) {
        int len1 = s1.length();
        int len2 = s2.length();
        
        // Если s1 длиннее s2, то s1 не может быть подстрокой s2
        if (len1 > len2) return false;
        
        // Создаём два массива для подсчёта частот символов
        // Каждый массив размером 26 (для 26 букв английского алфавита)
        int[] count1 = new int[26]; // для шаблона s1
        int[] count2 = new int[26]; // для скользящего окна в s2
        
        // Заполняем count1 частотами символов из s1
        // И одновременно заполняем count2 для первого окна размером len1 в s2
        for (int i = 0; i < len1; i++) {
            count1[s1.charAt(i) - 'a']++; // увеличиваем счётчик для символа из s1
            count2[s2.charAt(i) - 'a']++; // увеличиваем счётчик для символа из s2
        }
        
        // Проверяем, совпадают ли частоты для первого окна
        if (matches(count1, count2)) {
            return true;
        }
        
        // Используем технику скользящего окна
        // Перемещаем окно размером len1 по строке s2
        for (int i = len1; i < len2; i++) {
            // Добавляем новый символ в окно (правый указатель)
            count2[s2.charAt(i) - 'a']++;
            // Удаляем старый символ из окна (левый указатель)
            count2[s2.charAt(i - len1) - 'a']--;
            
            // Проверяем, совпадают ли частоты после обновления окна
            if (matches(count1, count2)) {
                return true;
            }
        }
        
        return false;
    }
    
    // Вспомогательная функция для проверки совпадения частот
    private boolean matches(int[] count1, int[] count2) {
        for (int i = 0; i < 26; i++) {
            // Если частоты какого-либо символа не совпадают, возвращаем false
            if (count1[i] != count2[i]) {
                return false;
            }
        }
        return true;
    }
}
```



## Временная сложность:
Временная сложность: O(n), где n - длина строки s2

## Пространственная сложность:
O(1)
