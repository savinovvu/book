## О чём задача?

Дан массив строк strs. Нужно сгруппировать анаграммы вместе. Можно возвращать ответ в любом порядке.

Анаграмма - это слово или фраза, образованная перестановкой букв другого слова или фразы.

Пример:

text
Input: strs = ["eat","tea","tan","ate","nat","bat"]
Output: [["bat"],["nat","tan"],["ate","eat","tea"]]

Input: strs = [""]
Output: [[""]]

Input: strs = ["a"]
Output: [["a"]]


## Решение на Java

```java
class Solution {
    public List<List<String>> groupAnagrams(String[] strs) {
        // Создаем хеш-таблицу для хранения групп анаграмм
        // Ключ - отсортированная строка, значение - список исходных строк
        Map<String, List<String>> map = new HashMap<>();
        
        // Проходим по всем строкам в массиве
        for (String s : strs) {
            // Преобразуем строку в массив символов
            char[] chars = s.toCharArray();
            // Сортируем символы в массиве
            Arrays.sort(chars);
            // Создаем ключ из отсортированного массива символов
            String key = new String(chars);
            
            // Если ключа еще нет в карте, создаем новый список для этой группы
            if (!map.containsKey(key)) {
                map.put(key, new ArrayList<>());
            }
            // Добавляем исходную строку в список, соответствующий ключу
            map.get(key).add(s);
        }
        
        // Возвращаем все значения карты в виде списка списков
        return new ArrayList<>(map.values());
    }
}
```


## Временная сложность:
Временная сложность: O(n * k log k)

n - количество строк

k - максимальная длина строки

Сортировка каждой строки: O(k log k)

## Пространственная сложность:
O(n * k)
