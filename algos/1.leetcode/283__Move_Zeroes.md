## О чём задача?

Задача "Move Zeroes" (Перемещение нулей) требует переместить все нули в массиве в его конец, сохраняя относительный порядок ненулевых элементов.

Условия:

Нельзя создавать копию массива

Необходимо минимизировать общее количество операций

Относительный порядок ненулевых элементов должен сохраниться


## Решение на Java

```java
class Solution {
    public void moveZeroes(int[] nums) {
        // Указатель для записи ненулевых элементов
        int nonZeroPointer = 0;
        
        for (int i = 0; i < nums.length; i++) {
            // Если текущий элемент не равен нулю
            if (nums[i] != 0) {
                // Меняем местами текущий элемент и элемент на позиции nonZeroPointer
                int temp = nums[nonZeroPointer];
                nums[nonZeroPointer] = nums[i];
                nums[i] = temp;
                // Увеличиваем указатель
                nonZeroPointer++;
            }
        }
    }
}
```
