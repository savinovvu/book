## О чём задача?

Даны две строки s и t. Нужно определить, являются ли они изоморфными.

Изоморфные строки - это строки, в которых символы первой строки могут быть заменены на символы второй строки с сохранением порядка, при этом:

Все вхождения одного символа должны заменяться на один и тот же символ

Разные символы не могут заменяться на один и тот же символ

Соответствие между символами должно быть взаимно однозначным

Примеры:

text
s = "egg", t = "add" → true
e → a, g → d

s = "foo", t = "bar" → false
f → b, o → a, o → r (нарушение: o заменяется на разные символы)

s = "paper", t = "title" → true
p → t, a → i, e → l, r → e

s = "badc", t = "baba" → false
b → b, a → a, d → b (нарушение: разные символы d и a заменяются на один символ b)


## Решение на Java

```java
class Solution {
    public boolean isIsomorphic(String s, String t) {
        // Если строки разной длины, они не могут быть изоморфными
        if (s.length() != t.length()) {
            return false;
        }
        
        // Карта для хранения соответствия символов из s в t
        Map<Character, Character> charMapping = new HashMap<>();
        // Множество для отслеживания уже использованных символов в t
        Set<Character> usedChars = new HashSet<>();
        
        // Проходим по всем символам обеих строк
        for (int i = 0; i < s.length(); i++) {
            char charS = s.charAt(i);
            char charT = t.charAt(i);
            
            // Если символ из s уже встречался
            if (charMapping.containsKey(charS)) {
                // Проверяем, соответствует ли он тому же символу в t
                if (charMapping.get(charS) != charT) {
                    return false; // Нарушение: один символ s соответствует разным символам t
                }
            } else {
                // Если символ из s встречается впервые
                // Проверяем, не использован ли уже символ t для другого символа s
                if (usedChars.contains(charT)) {
                    return false; // Нарушение: разные символы s соответствуют одному символу t
                }
                
                // Добавляем новое соответствие
                charMapping.put(charS, charT);
                usedChars.add(charT);
            }
        }
        
        return true;
    }
}
```
