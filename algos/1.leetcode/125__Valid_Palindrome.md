## О чём задача?

Дана строка s. Нужно определить, является ли она палиндромом после преобразования всех заглавных букв в строчные и удаления всех не-алфавитно-цифровых символов.

Палиндром - строка, которая читается одинаково слева направо и справа налево.

Примеры:

text
Input: s = "A man, a plan, a canal: Panama"
Output: true
Объяснение: "amanaplanacanalpanama" - палиндром

Input: s = "race a car"
Output: false
Объяснение: "raceacar" - не палиндром

Input: s = " "
Output: true
Объяснение: Пустая строка - палиндром
## Решение на Java

```java
class Solution {
    public boolean isPalindrome(String s) {
        // Проверка на пустую строку - по условию пустая строка считается палиндромом
        if (s.isEmpty()) {
            return true;
        }
        
        // Инициализируем два указателя:
        // left начинает с начала строки (индекс 0)
        // right начинает с конца строки (последний индекс)
        int left = 0;
        int right = s.length() - 1;
        
        // Продолжаем пока указатели не встретятся
        while (left <= right) {
            // Получаем символы на текущих позициях указателей
            char leftChar = s.charAt(left);
            char rightChar = s.charAt(right);
            
            // Если левый символ не является буквой или цифрой
            if (!Character.isLetterOrDigit(leftChar)) {
                left++; // пропускаем этот символ, двигаем левый указатель вперед
            }
            // Если правый символ не является буквой или цифрой
            else if (!Character.isLetterOrDigit(rightChar)) {
                right--; // пропускаем этот символ, двигаем правый указатель назад
            }
            // Оба символа являются буквенно-цифровыми
            else {
                // Сравниваем символы в нижнем регистре
                if (Character.toLowerCase(leftChar) != Character.toLowerCase(rightChar)) {
                    return false; // если не совпадают - строка не палиндром
                }
                // Символы совпали, двигаем оба указателя
                left++;
                right--;
            }
        }
        
        // Если дошли до этого места, все символы совпали - строка палиндром
        return true;
    }
}
```
