## О чём задача?

Задача "One Edit Distance" (Расстояние в одно редактирование) проверяет, 
можно ли превратить одну строку в другую ровно за одну операцию редактирования.

Операции редактирования:

Вставка одного символа
Удаление одного символа
Замена одного символа

Примеры:

"abc" и "ab" → True (удаление 'c')
"abc" и "abd" → True (замена 'c' на 'd')
"abc" и "abcd" → True (вставка 'd')
"abc" и "abc" → False (одинаковые строки)
"abc" и "ade" → False (нужно больше одной операции)


## Решение на Java

```java
class Solution {
    public boolean isOneEditDistance(String s, String t) {
        int m = s.length();
        int n = t.length();
        
        // Стандартизируем: делаем s более короткой строкой
        // Это упрощает логику, так как мы всегда работаем с более короткой строкой
        if (m > n) {
            return isOneEditDistance(t, s);
        }
        
        // Если разница в длине больше 1, невозможно преобразовать за одну операцию
        // (максимальная разница для одной операции вставки/удаления = 1)
        if (n - m > 1) {
            return false;
        }
        
        // Проходим по символам более короткой строки
        for (int i = 0; i < m; i++) {
            // Находим первое различие между строками
            if (s.charAt(i) != t.charAt(i)) {
                if (m == n) {
                    // Случай 1: длины строк равны
                    // Единственная возможная операция - замена символа
                    // Проверяем, что оставшиеся части строк (после текущей позиции) идентичны
                    return s.substring(i + 1).equals(t.substring(i + 1));
                } else {
                    // Случай 2: длины строк отличаются на 1
                    // Единственная возможная операция - вставка/удаление
                    // Пропускаем один символ в более длинной строке и проверяем совпадение
                    return s.substring(i).equals(t.substring(i + 1));
                }
            }
        }
        
        // Если мы дошли до конца без нахождения различий в символах,
        // то единственная возможная операция - добавить один символ в конец более короткой строки
        // Это возможно только если разница в длине = 1
        // если разницы в длинне нет, то строки идентичны и это false
        return m + 1 == n;
    }
}
```
