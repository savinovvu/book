## О чём задача?

5. Longest Palindromic Substring


## Решение на Java

```java
public class Solution {
    
    /**
     * Основной метод для поиска самой длинной палиндромной подстроки
     */
    public String longestPalindrome(String s) {
        if (s == null || s.length() < 1) {
            return "";
        }
        
        int start = 0; // Начальный индекс самой длинной палиндромной подстроки
        int end = 0;   // Конечный индекс самой длинной палиндромной подстроки
        
        // Проходим по каждому символу строки
        for (int i = 0; i < s.length(); i++) {
            // Случай 1: палиндром с центром в одном символе (нечётная длина)
            int len1 = expandAroundCenter(s, i, i);
            // Случай 2: палиндром с центром между двумя символами (чётная длина)
            int len2 = expandAroundCenter(s, i, i + 1);
            
            // Выбираем максимальную длину из двух случаев
            int len = Math.max(len1, len2);
            
            // Если найденная длина больше текущей максимальной
            if (len > end - start) {
                // Пересчитываем начальный и конечный индексы
                // Для нечётного палиндрома: start = i - (len-1)/2
                // Для чётного палиндрома: та же формула работает корректно
                start = i - (len - 1) / 2;
                end = i + len / 2;
            }
        }
        
        // Возвращаем подстроку от start до end (включительно)
        return s.substring(start, end + 1);
    }
    
    /**
     * Вспомогательный метод для расширения от центра
     * @param s исходная строка
     * @param left левый индекс центра
     * @param right правый индекс центра
     * @return длина палиндрома
     */
    private int expandAroundCenter(String s, int left, int right) {
        // Пока символы слева и справа совпадают и не вышли за границы строки
        while (left >= 0 && right < s.length() && s.charAt(left) == s.charAt(right)) {
            left--;  // Расширяемся влево
            right++; // Расширяемся вправо
        }
        
        // Возвращаем длину палиндрома
        // После выхода из цикла left и right указывают на символы за пределами палиндрома
        // Поэтому длина = (right - 1) - (left + 1) + 1 = right - left - 1
        return right - left - 1;
    }
}
```


## Временная сложность:
O(n²)

## Пространственная сложность:
O(1)
