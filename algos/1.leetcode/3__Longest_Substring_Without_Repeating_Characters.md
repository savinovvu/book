## О чём задача?
Задача "Longest Substring Without Repeating Characters" (Самая длинная подстрока без повторяющихся символов) 

требует найти длину самой длинной подстроки в заданной строке, 
в которой все символы уникальны.

## Решение на Java

```java
class Solution {
    public int lengthOfLongestSubstring(String s) {
        // Создаем хэш-таблицу для отслеживания последнего индекса каждого символа
        // Ключ: символ, Значение: последний индекс, где он встречался
        Map<Character, Integer> charIndexMap = new HashMap<>();
        
        int maxLength = 0; // Переменная для хранения максимальной длины
        int left = 0;      // Левый указатель скользящего окна
        
        // Правый указатель проходит по всей строке
        for (int right = 0; right < s.length(); right++) {
            char currentChar = s.charAt(right);
            
            // Проверяем, встречался ли текущий символ в текущем окне
            if (charIndexMap.containsKey(currentChar)) {
                // Если символ уже есть в окне, сдвигаем левую границу
                // за последнее вхождение этого символа
                // Math.max гарантирует, что left не сдвинется назад
                // (защита от случаев, когда символ встречался раньше текущего окна)
                left = Math.max(charIndexMap.get(currentChar) + 1, left);
            }
            
            // Обновляем индекс текущего символа в хэш-таблице
            charIndexMap.put(currentChar, right);
            
            // Вычисляем длину текущего окна и обновляем максимум
            // right - left + 1 = длина текущей подстроки
            maxLength = Math.max(maxLength, right - left + 1);
        }
        
        return maxLength;
    }
}
```


## Подробные комментарии к решению


## Временная сложность:
Временная сложность: O(n)
Мы проходим по строке только один раз с помощью правого указателя

Каждая операция с HashMap (put, get, containsKey) занимает O(1) в среднем случае

Общее время: O(n), где n - длина строки

## Пространственная сложность:

Пространственная сложность: O(min(m, n))
m - размер алфавита (количество уникальных символов)

n - длина строки

В худшем случае храним все уникальные символы строки

Для ASCII максимально 128 символов, для Unicode больше, но ограничено
