## О чём задача?

О задаче
Задача "Find K Closest Elements" (Найти K ближайших элементов) заключается в следующем:
Дан отсортированный массив целых чисел arr, целое число k и целое число x. 
Нужно найти k ближайших элементов к числу x в массиве. 
Результат должен быть отсортирован по возрастанию.

Важные детали:
Если два элемента находятся на одинаковом расстоянии от x, предпочтение отдается меньшему элементу
Результат должен быть отсортирован по возрастанию
Массив уже отсортирован по возрастанию


## Решение на Java

```java
class Solution {
    public List<Integer> findClosestElements(int[] arr, int k, int x) {
        // Бинарный поиск для нахождения позиции, где можно вставить x
        int left = 0; // минимально возможный старт окна
        int right = arr.length - k; // Максимально возможный старт окна
        
        while (left < right) {
            // Находим середину текущего диапазона
            int mid = left + (right - left) / 2;

            // Сравниваем, какое окно лучше:
            // окно [mid, mid + k - 1] или [mid + 1, mid + k]
            // Ключевая идея: сравниваем расстояние от x до левой границы
            // и до элемента сразу за правой границей
            
            if (x - arr[mid] > arr[mid + k] - x) {
                // Если расстояние от x до arr[mid] больше, чем до arr[mid + k],
                // значит окно [mid, mid + k - 1] не оптимально - сдвигаем вправо
                left = mid + 1;
            } else {
                // Иначе окно [mid, mid + k - 1] может быть оптимальным
                // или мы можем найти лучшее окно левее
                right = mid;
            }
        }

        // После бинарного поиска left указывает на оптимальную стартовую позицию
        // Формируем результат из k элементов, начиная с позиции left
        List<Integer> result = new ArrayList<>();
        for (int i = left; i < left + k; i++) {
            result.add(arr[i]);
        }
        
        return result;
    }
}
```


## Подробные комментарии к решению
Сравнение расстояний - ключевая логика для принятия решения о сдвиге окна
Условие предпочтения меньшего элемента автоматически выполняется при движении окна влево

## Временная сложность:
O(log(n - k) + k)

## Пространственная сложность:
O(1)
