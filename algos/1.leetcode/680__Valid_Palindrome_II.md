## О чём задача?

Дана строка s. Можно удалить не более одного символа. Проверить, может ли строка стать палиндромом после такого удаления.

Палиндром — это строка, которая читается одинаково слева направо и справа налево.

Примеры:

"aba" → true (уже палиндром)

"abca" → true (можно удалить 'c', получится "aba")

"abc" → false (ни одно удаление не даст палиндром)

## Решение на Java

```java
class Solution {
    public boolean validPalindrome(String s) {
        // Используем технику двух указателей:
        // left начинает с начала строки (индекс 0)
        // right начинает с конца строки (индекс s.length()-1)
        int left = 0;
        int right = s.length() - 1;
        
        // Продолжаем пока указатели не встретятся в центре
        while (left < right) {
            // Если находим несовпадающие символы
            if (s.charAt(left) != s.charAt(right)) {
                // У нас есть два варианта:
                // 1. Удалить символ слева (пропустить left) и проверить оставшуюся подстроку
                // 2. Удалить символ справа (пропустить right) и проверить оставшуюся подстроку
                // Если хотя бы один вариант работает - возвращаем true
                return isPalindrome(s, left + 1, right) || isPalindrome(s, left, right - 1);
            }
            // Если символы совпадают, двигаемся к центру
            left++;
            right--;
        }
        
        // Если прошли всю строку без несовпадений - это палиндром
        return true;
    }
    
    // Вспомогательный метод для проверки палиндрома в указанном диапазоне
    // Проверяет подстроку s[left...right] (включительно)
    private boolean isPalindrome(String s, int left, int right) {
        // Используем ту же технику двух указателей для проверки подстроки
        while (left < right) {
            // Если нашли несовпадение - не палиндром
            if (s.charAt(left) != s.charAt(right)) {
                return false;
            }
            // Двигаем указатели к центру
            left++;
            right--;
        }
        // Все символы совпали - это палиндром
        return true;
    }
}
```
