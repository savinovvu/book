## О чём задача?

Дано целое число n. 
Нужно найти минимальное количество идеальных квадратов (1, 4, 9, 16, ...), сумма которых равна n.

Идеальный квадрат - это целое число, которое является квадратом другого целого числа.

Примеры:

text
Input: n = 12
Output: 3
Объяснение: 12 = 4 + 4 + 4

Input: n = 13
Output: 2
Объяснение: 13 = 4 + 9

## Решение на Java

```java
class Solution {
    public int numSquares(int n) {
        // Проверяем, является ли число полным квадратом
        if (isSquare(n)) return 1;
        
        // Проверяем, можно ли представить как сумму двух квадратов
        for (int i = 1; i * i <= n; i++) {
            if (isSquare(n - i * i)) return 2;
        }
        
        // Проверяем по теореме Лагранжа, 3 квадрата, если n не имеет вид n = 4^a × (8b + 7)
        while (n % 4 == 0) n /= 4;
        if (n % 8 == 7) return 4;
        
        // Остается вариант с тремя квадратами
        return 3;
    }
    
    private boolean isSquare(int n) {
        int sqrt = (int) Math.sqrt(n);
        return sqrt * sqrt == n;
    }
}
```
