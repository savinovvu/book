## О чём задача?

Даны два узла p и q в бинарном дереве. Нужно найти их наименьшего общего предка (LCA).

Особенность: У каждого узла есть указатель на родителя (parent), но нет указателей на левого и правого потомка.

Определение LCA: Наименьший общий предок двух узлов p и q - это самый нижний узел в дереве, который является предком как p, так и q.

Пример:

             3
           /   \
          5     1
         / \   / \
        6   2 0   8
           / \
          7   4

Input: p = 5, q = 1
Output: 3

Input: p = 5, q = 4

## Решение на Java

```java
class Node {
    public int val;
    public Node left;
    public Node right;
    public Node parent;
}

class Solution {
    public Node lowestCommonAncestor(Node p, Node q) {
        // Используем два указателя, аналогично задаче о пересечении связанных списков
        Node a = p;
        Node b = q;
        
        // Двигаемся от обоих узлов к корню
        // Когда один достигает конца, переключаем на другой узел
        // Это создает эффект "цикла", 
        // что компенсирует разницу в длинах путей
        // и указатели встретятся в LCA
        
        while (a != b) {
            a = (a == null) ? q : a.parent;
            b = (b == null) ? p : b.parent;
        }
        
        return a; // точка встречи - это LCA
    }
}
```
