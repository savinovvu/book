## О чём задача?

Даны два списка закрытых интервалов firstList и secondList, 
где каждый интервал [start, end]. 
Нужно найти пересечение этих двух списков интервалов.

Пересечение интервалов - это множество вещественных чисел, которые принадлежат обоим интервалам.

Пример:

text
Input: firstList = [[0,2],[5,10],[13,23],[24,25]],
secondList = [[1,5],[8,12],[15,24],[25,26]]
Output: [[1,2],[5,5],[8,10],[15,23],[24,24],[25,25]]
## Решение на Java

```java
class Solution {
    public int[][] intervalIntersection(int[][] firstList, int[][] secondList) {
        // Если один из списков пуст, пересечений нет
        if (firstList.length == 0 || secondList.length == 0) {
            return new int[0][0];
        }
        
        List<int[]> result = new ArrayList<>();
        int i = 0; // указатель для первого списка
        int j = 0; // указатель для второго списка
        
        // Проходим по обоим спискам одновременно
        while (i < firstList.length && j < secondList.length) {
            // Текущие интервалы из обоих списков
            int[] first = firstList[i];
            int[] second = secondList[j];
            
            // Находим потенциальное пересечение:
            // начало - максимум из начал двух интервалов
            // конец - минимум из концов двух интервалов
            int start = Math.max(first[0], second[0]);
            int end = Math.min(first[1], second[1]);
            
            // Если начало <= концу, значит есть валидное пересечение
            if (start <= end) {
                result.add(new int[]{start, end});
            }
            
            // Перемещаем указатель того списка, у которого интервал заканчивается раньше
            // Это потому что следующий интервал в этом списке может еще пересекаться
            // с текущим интервалом из другого списка
            if (first[1] < second[1]) {
                i++; // переходим к следующему интервалу в первом списке
            } else {
                j++; // переходим к следующему интервалу во втором списке
            }
        }
        
        // Преобразуем список в массив для возврата
        return result.toArray(new int[result.size()][]);
    }
}
```


## Подробные комментарии к решению


## Временная сложность:
O(m + n)
m - длина firstList
n - длина secondList

Каждый интервал обрабатывается один раз

## Пространственная сложность:
O(m + n)
