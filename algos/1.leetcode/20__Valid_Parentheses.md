## О чём задача?

Дана строка s, содержащая только символы: '(', ')', '{', '}', '[', ']'. Нужно определить, является ли строка валидной.

Строка валидна, если:

Открытые скобки должны быть закрыты скобками того же типа

Открытые скобки должны быть закрыты в правильном порядке

Каждая закрывающая скобка имеет соответствующую открывающую скобку того же типа

## Решение на Java

```java
class Solution {
    public boolean isValid(String s) {
        // Используем стек для отслеживания открывающих скобок
        Stack<Character> stack = new Stack<>();
        
        // Проходим по каждому символу в строке
        for (char c : s.toCharArray()) {
            // Если символ - открывающая скобка, добавляем её в стек
            if (c == '(' || c == '[' || c == '{') {
                stack.push(c);
            } 
            // Если символ - закрывающая скобка
            else {
                // Если стек пуст, значит нет соответствующей открывающей скобки
                if (stack.isEmpty()) {
                    return false;
                }
                
                // Извлекаем последнюю открывающую скобку из стека
                char top = stack.pop();
                
                // Проверяем, соответствует ли закрывающая скобка открывающей
                if ((c == ')' && top != '(') ||
                    (c == ']' && top != '[') ||
                    (c == '}' && top != '{')) {
                    return false;
                }
            }
        }
        
        // Если после обработки всех символов стек пуст, значит все скобки закрыты правильно
        // Если стек не пуст, значит есть незакрытые скобки
        return stack.isEmpty();
    }
}
```
