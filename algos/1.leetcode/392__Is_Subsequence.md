## О чём задача?

Условие задачи:
Даны две строки s и l. 
Нужно определить, является ли s подпоследовательностью l.

Подпоследовательность - это последовательность, 
которая может быть получена из другой последовательности 
удалением некоторых элементов без изменения порядка оставшихся элементов.


## Решение на Java

```java
class Solution {
    public boolean isSubsequence(String s, String l) {
        // Используем два указателя:
        // i - отслеживает текущую позицию в строке s (что мы ищем)
        // j - отслеживает текущую позицию в строке l (где мы ищем)
        int i = 0;
        int j = 0;
        
        // Продолжаем поиск, пока не дойдем до конца одной из строк
        while (i < s.length() && j < l.length()) {
            // Если нашли совпадение символов
            if (s.charAt(i) == l.charAt(j)) {
                i++; // переходим к следующему символу в s
                // Теперь будем искать этот следующий символ в оставшейся части l
            }
            // Независимо от того, было совпадение или нет,
            // всегда двигаемся вперед по строке l
            j++;
        }
        
        // Проверяем, дошли ли мы до конца строки s
        // Если да - значит все символы s были найдены в l в правильном порядке
        // Если нет - значит некоторые символы из s не были найдены
        return i == s.length();
    }
}
```
