## О чём задача?

Дан массив целых чисел nums. Нужно вернуть массив answer такой, что answer[i] равен произведению всех элементов nums, кроме nums[i].

Ограничения:

Нельзя использовать операцию деления

Должно работать за O(n) время

Пространственная сложность: O(1) (не считая выходного массива)

Пример:

text
Input: nums = [1,2,3,4]
Output: [24,12,8,6]

Input: nums = [-1,1,0,-3,3]
Output: [0,0,9,0,0]


## Решение на Java

```java
class Solution {
    public int[] productExceptSelf(int[] nums) {
        int n = nums.length;
        int[] answer = new int[n];
        
        // Первый проход: вычисляем произведения всех элементов слева от текущего
        // answer[i] будет содержать произведение всех элементов от nums[0] до nums[i-1]
        answer[0] = 1; // Для первого элемента нет элементов слева, поэтому произведение = 1
        
        for (int i = 1; i < n; i++) {
            answer[i] = answer[i - 1] * nums[i - 1];
        }
        
        // Второй проход: вычисляем произведения всех элементов справа от текущего
        // и умножаем на уже имеющиеся произведения слева
        int rightProduct = 1; // Начинаем с 1 для последнего элемента
        
        for (int i = n - 1; i >= 0; i--) {
            // Умножаем текущее значение (произведение слева) на произведение справа
            answer[i] = answer[i] * rightProduct;
            // Обновляем произведение справа для следующей итерации
            rightProduct *= nums[i];
        }
        
        return answer;
    }
}
```
