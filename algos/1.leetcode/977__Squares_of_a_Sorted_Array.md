## О чём задача?

Вам дан массив целых чисел, отсортированный в неубывающем порядке (от меньшего к большему). 

Вам нужно вернуть новый массив, состоящий из квадратов каждого числа, 

также отсортированный в неубывающем порядке.

Ключевая сложность: Исходный массив может содержать отрицательные числа. 

Когда отрицательное число возводится в квадрат, оно становится положительным, 

поэтому простого возведения в квадрат и сортировки недостаточно для максимальной эффективности.


## Решение на Java

```java
class Solution {
    public int[] sortedSquares(int[] nums) {
        int n = nums.length;
        // Создаём массив для результата такого же размера
        int[] result = new int[n];
        
        // Два указателя: 
        // left - начинает с начала массива (отрицательные числа)
        // right - начинает с конца массива (положительные числа)
        int left = 0;
        int right = n - 1;
        
        // Заполняем массив result с конца, так как наибольшие квадраты 
        // будут у крайних элементов (из-за отрицательных чисел)
        for (int i = n - 1; i >= 0; i--) {
            // Вычисляем квадраты элементов на указателях
            int leftSquare = nums[left] * nums[left];
            int rightSquare = nums[right] * nums[right];
            
            // Сравниваем квадраты и выбираем больший
            if (leftSquare > rightSquare) {
                // Если квадрат левого элемента больше, помещаем его в результат
                result[i] = leftSquare;
                left++; // Сдвигаем левый указатель, так как этот элемент уже обработан
            } else {
                // Если квадрат правого элемента больше или равен, помещаем его
                result[i] = rightSquare;
                right--; // Сдвигаем правый указатель
            }
        }
        
        return result;
    }
}
```
