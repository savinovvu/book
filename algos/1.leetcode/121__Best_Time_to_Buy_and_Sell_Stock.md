## О чём задача?

Дан массив prices, где prices[i] представляет цену акции в i-й день. 
Нужно найти максимальную прибыль, которую можно получить,
купив акции в один день и продав в другой день в будущем. 
Если прибыль невозможна, вернуть 0.

Ограничения:

Можно совершить только одну транзакцию (купить один раз и продать один раз)

День покупки должен быть раньше дня продажи

Примеры:

text
Input: prices = [7,1,5,3,6,4]
Output: 5
Объяснение: Купить в день 1 (цена=1) и продать в день 4 (цена=6), прибыль = 6-1 = 5

Input: prices = [7,6,4,3,1]
Output: 0
Объяснение: Невозможно получить прибыль, цены только падают


## Решение на Java

```java
class Solution {
    public int maxProfit(int[] prices) {
        if (prices == null || prices.length == 0) {
            return 0;
        }
        
        int minPrice = Integer.MAX_VALUE; // Минимальная цена для покупки
        int maxProfit = 0;                // Максимальная прибыль
        
        for (int price : prices) {
            minPrice = Math.min(price, minPrice);
            // Рассчитываем потенциальную прибыль и обновляем максимум
            maxProfit = Math.max(price - minPrice, maxProfit);
        }
        
        return maxProfit;
    }
}
```


## Временная сложность:
O(n)

## Пространственная сложность:
O(1)
