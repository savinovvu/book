## О чём задача?

О задаче "Rotting Oranges"
Условие задачи:
Дан двумерный массив grid размером m × n, где каждая ячейка может иметь одно из трех значений:

0 - пустая ячейка

1 - свежий апельсин

2 - гнилой апельсин

Каждую минуту любой свежий апельсин, соседний с гнилым (по вертикали или горизонтали), становится гнилым.

Нужно найти минимальное количество минут, через которое все апельсины станут гнилыми. Если это невозможно, вернуть -1.

Примеры:

text
Input: grid = [[2,1,1],[1,1,0],[0,1,1]]
Output: 4
Объяснение: Процесс гниения занимает 4 минуты

Input: grid = [[2,1,1],[0,1,1],[1,0,1]]
Output: -1
Объяснение: Апельсин в левом нижнем углу никогда не сгниет


## Решение на Java

```java
class Solution {
    public int orangesRotting(int[][] grid) {
        if (grid == null || grid.length == 0) return 0;
        
        int rows = grid.length;
        int cols = grid[0].length;
        
        // Очередь для хранения позиций гнилых апельсинов
        Queue<int[]> queue = new LinkedList<>();
        int freshOranges = 0; // Счётчик свежих апельсинов
        
        // Проходим по всей сетке, чтобы:
        // 1. Найти все изначально гнилые апельсины и добавить их в очередь
        // 2. Посчитать количество свежих апельсинов
        for (int i = 0; i < rows; i++) {
            for (int j = 0; j < cols; j++) {
                if (grid[i][j] == 2) {
                    queue.offer(new int[]{i, j}); // Добавляем координаты гнилого апельсина
                } else if (grid[i][j] == 1) {
                    freshOranges++; // Увеличиваем счётчик свежих апельсинов
                }
            }
        }
        
        // Если изначально нет свежих апельсинов, то время = 0
        if (freshOranges == 0) return 0;
        
        int minutes = 0;
        // Направления для проверки соседних клеток: вверх, вниз, влево, вправо
        int[][] directions = {{1, 0}, {-1, 0}, {0, 1}, {0, -1}};
        
        // BFS (поиск в ширину) для распространения гниения
        while (!queue.isEmpty()) {
            int size = queue.size();
            boolean rottenThisMinute = false; // Флаг, были ли новые гниения в эту минуту
            
            // Обрабатываем все гнилые апельсины на текущей минуте
            for (int i = 0; i < size; i++) {
                int[] current = queue.poll(); // Извлекаем текущий гнилой апельсин
                int x = current[0];
                int y = current[1];
                
                // Проверяем всех 4 соседей
                for (int[] dir : directions) {
                    int dx = x + dir[0];
                    int dy = y + dir[1];
                    
                    // Если соседняя клетка в пределах сетки и содержит свежий апельсин
                    if (dx >= 0 && dx < rows && dy >= 0 && dy < cols && 
                        grid[dx][dy] == 1) {
                        
                        // Делаем апельсин гнилым
                        grid[dx][dy] = 2;
                        // Добавляем новую позицию гнилого апельсина в очередь
                        queue.offer(new int[]{dx, dy});
                        // Уменьшаем счётчик свежих апельсинов
                        freshOranges--;
                        // Устанавливаем флаг, что в эту минуту было гниение
                        rottenThisMinute = true;
                    }
                }
            }
            
            // Если в эту минуту были новые гниения, увеличиваем счётчик минут
            if (rottenThisMinute) {
                minutes++;
            }
        }
        
        // Если остались свежие апельсины, возвращаем -1, иначе возвращаем общее время
        return freshOranges == 0 ? minutes : -1;
    }
}
```
